## 🛠 Methodology & Glossary

### Metric Definitions
* **RER (Reach Efficiency Ratio):** `Views / Followers`. Used to determine how much "free" distribution the algorithm grants a piece of content relative to the creator's base.
* **SVR (Save-to-View Ratio):** `Saves / Views`. A primary indicator of "Wishlist Intent" and high-utility content.
* **Balanced Reviews:** Content mentioning minor drawbacks. Our data shows these capture **2.3x more shares** than "Pure Rave" scripted enthusiasm.
